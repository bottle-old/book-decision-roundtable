## 项目简介

**百书人生决策智库圆桌**是一套基于100本精选书籍构建的AI决策辅助Skill。不同于传统的"AI读书笔记"——本项目采用**主持人控场+书籍导师轮流发言**的圆桌对话形式，所有分析内容均为**原创复述与独立撰写**，不包含任何原书原文引用。

### 三大核心特色

| 特色 | 说明 |
|------|------|
| 🎙️ 主持人控场×轮流发言 | 固定专属主持人+4位导师顺序单人完整发言 |
| 📐 双维度分析 | 时间推演（当下/1-3年/5-10年）+ 分领域落地实操 |
| 📚 百书智库体系 | 100本藏书覆盖四大领域，条条观点可溯源至书籍编号 |

## 本地使用教程

### Step 1：加载Skill核心逻辑
将 `skill_core/roundtable_main.md` 和 `skill_core/rule_limit.md` 的内容作为AI Agent的系统提示词。

### Step 2：描述困境
向AI Agent描述你当前面临的两难抉择（包含年龄、财务、技能、可选路线）。

### Step 3：获得圆桌分析
主持人自动匹配4位书籍导师 → 轮流发言 → 输出双方案（保守稳妥+成长突破）。
