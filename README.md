# 📚 百书人生决策智库圆桌

> 当你站在人生十字路口——**4位书籍导师围坐一圈，轮流为你拆解每条路该怎么走。**

---

## 🎯 一句话说清楚

你向AI描述一个人生两难困境（换行业还是深耕？创业还是稳住？），AI主持人自动从100本书中匹配4位最相关的"书籍导师"，按圆桌对话形式轮流发言，最终给你两套**立刻能执行**的方案——一套稳妥的，一套突破的。

---

## ✨ 和别的"AI读书"有什么不同

| 别人 | 本项目 |
|------|--------|
| AI朗读一本书的摘要 | 4位导师**跨领域围桌对话**，互补视角 |
| 给一堆理论让你自己消化 | 每条建议都带**实操步骤、资源需求、执行难度** |
| 自由辩论，越聊越散 | **主持人全程控场**，一位讲完下一位再接，不跑题 |
| 引用互联网上的碎片信息 | 所有观点来自100本精选藏书，**条条可溯源至书籍编号** |
| 只谈当下怎么做 | **当下 → 1-3年 → 5-10年** 三层时间推演 |把落地实操加上时间线可以一笔带过

## 🎙️ 三大核心机制

### 1. 主持人控场 × 导师轮流发言
每场圆桌由专属主持人开场拆解困境 → 检索藏书库匹配4位导师 → 导师**按顺序单人完整发言**（一人讲完再换下一位）→ 主持人收尾输出双方案。不走辩论、不对抗，专注**出方案**。

### 2. 双维度深度分析
每位导师的发言同时覆盖两个维度：
- **时间推演**：当下成本与风险 / 1-3年发展瓶颈 / 5-10年天花板与容错空间
- **本领域落地**：分阶段执行步骤 + 资金/时间/技能/人脉资源清单 + 高频踩坑点与规避手段

### 3. 百书分领域智库
100本书覆盖四大领域，跨领域导师组合保证视角不重叠：

| 领域 | 代表书目 | 擅长解决的困境 |
|------|---------|--------------|
| 💰 金融理财 | 《穷查理宝典》《经济学原理》《段永平投资问答录》… | 投资决策、资产配置、创业融资、股权设计 |
| 📱 短视频&广告运营 | 《定位》《影响力》《增长黑客》《疯传》… | 品牌定位、用户增长、内容策略、直播电商 |
| 🎓 个人成长&学历考证 | 《高效能人士的七个习惯》《金字塔原理》《认知觉醒》… | 职业规划、学习效率、时间管理、目标设定 |
| 🧠 底层认知思维 | 《从0到1》《精益创业》《创新者的解答》《创业维艰》… | 创业方向、商业模式、创新策略、逆境决策 |

## 🚀 快速开始

**Step 1**：将 `skill_core/` 中两份核心文档作为AI Agent的系统提示词。

**Step 2**：向AI描述你的困境（年龄、财务、技能、可选路线、长期目标）。

**Step 3**：AI主持人自动匹配4位导师 → 轮流发言 → 输出两套可落地方案。

```
"我今年30岁，做了6年Java后端，存款30万，未婚。
该全力转型AI，还是深耕成为架构师？"
          ↓
主持人检索索引 → 匹配《从0到1》《穷查理宝典》《创新者的解答》《认知觉醒》
          ↓
4位导师轮流发言 → 保守稳妥方案 + 成长突破方案
```
